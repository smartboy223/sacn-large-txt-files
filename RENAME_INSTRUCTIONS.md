# Rename this repo for better discoverability

The README is already set up for the new name.

## Suggested name: **search-large-txt-files**

Why:
- People search for *"search large txt files"* or *"large text file search"*
- Hyphenated, lowercase, descriptive
- Easy to find on GitHub and Google

---

## Steps

1. **On GitHub:** Open your repo → **Settings** → under **Repository name** change  
   `sacn-large-txt-files` → **search-large-txt-files** → **Rename**.

2. **On your PC** (in this folder):
   ```bash
   git remote set-url origin https://github.com/smartboy223/search-large-txt-files.git
   git push origin main
   ```

3. **Add Topics** (repo page → ⚙️ next to "About"):  
   `search`, `text-search`, `txt`, `large-files`, `grep`, `local`, `nodejs`, `react`, `offline`

4. **Add a short description** (e.g.):  
   *Search across large .txt files with AND/OR/NOT — local, fast, no uploads.*

Done. The repo will be easier to find when others search for this kind of tool.
